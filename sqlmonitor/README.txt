Минимальные требования к операционной системе:
  Windows Vista.

Перечень файлов:
  sqlmonitor.exe - GUI утилита для отображения изменений в sqlsniffer.log файле
  sqlsniffer.exe - консольная утилита перехвата SQL-запросов и складирования их в sqlsniffer.log файл
  sqlsniffer.log - пример лог-файла
  WinDivert.dll   - часть свободной библиотеки перехвата TCP-пакетов в операционной среде Windows
  WinDivert32.sys - часть свободной библиотеки перехвата TCP-пакетов в операционной среде Windows
  WinDivert64.sys - часть свободной библиотеки перехвата TCP-пакетов в операционной среде Windows

Процесс использования:
1. В отдельном консольном окне запускаем sqlsniffer.exe с указанием исходящего порта для прослушивания.
   Пример: sqlsniffer.exe 1521
2. Запускаем sqlmonitor.exe, по кнопке [...] выбираем файл sqlsniffer.log, находящийся в директории, 
   из которого запущена утилита перехвата из пункта 1. Жмём кнопку [Start].
3. Открываем любимый клиент ORACLE и даём запросы к базе.
4. Наблюдаем обновление информации в sqlmonitor.exe
5. Открываем менее любимый клиент ORACLE и даём запросы к базе.
6. Наблюдаем обновление информации в sqlmonitor.exe
